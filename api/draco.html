Draco Helper Class#

Draco can check whether a visualization specification is valid. If it isnâ€™t valid, it can also compute what constrains are being violated. These methods check a specification against the standard set of constraints in Draco. If you define your own constraints, you can pass different ASP programs to the constructor or instead use the lower-level Draco run API.
